# My Portfolio

Welcome to my portfolio! This is a personal website built using React and Tailwind CSS to showcase my skills, work, experiences, and contact details. 

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly on mobile and desktop devices.
- **Skills Showcase**: Highlights my technical skills and expertise.
- **Project Portfolio**: A collection of my projects with descriptions and links.
- **Experience Section**: Details of my educational background and professional experience.
- **Contact Form**: A form for visitors to reach out to me directly.
